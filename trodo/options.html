TRODO

Link an account with a trade-only API key. TRODO never asks for your exchange password, PAN, or any ID document.

Before you link an account:
  1. On your exchange, create a new API key dedicated to TRODO.
  2. Enable Spot & Margin Trading only.
  3. Leave Enable Withdrawals switched off — TRODO will warn you if it detects it's on.
  4. Restrict the key to your IP if the exchange supports it.
